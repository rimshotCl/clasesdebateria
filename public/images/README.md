componente correspondiente si prefieres otros nombres):

| Archivo                              | Usado en                          | Descripción (alt text)                                   |
|---------------------------------------|------------------------------------|------------------------------------------------------------|
| hero-bateria.webp                    | Inicio                             | Profesor tocando batería durante una clase presencial      |

Formato: .webp — es más liviano que .jpg con la misma calidad visual,
por eso es el formato recomendado para todo el sitio. Idealmente cada
imagen debería pesar menos de 500kb.
